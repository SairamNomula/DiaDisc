### How to setup?
Instructions to set up DiaDisc locally.
To get a local copy up and running follow these simple steps.

### Prerequisites
You must have Python installed in your local system.
* flask
  ```sh
  pip install flask
  ```
  
* pickle
  ```sh
  pip install pickle
  ```
  
* numpy
  ```sh
  pip install numpy
  ```

* pyyaml
  ```sh
  pip install pyyaml
  ```

* flask_mysqldb
  ```sh
  pip install flask_mysqldb
  ```
